## Portfolio Rule

- Active repositories carry the main revenue story: enterprise AI, runtime reliability, security/ops, data governance, industrial AI, private deployment, and regulated workflows.
- Supporting and archived repositories stay monetizable as workshops, templates, diagnostics, revival packages, or optional domain breadth.
- Behavioral design is used for clarity, trust, and action. It should not be used for pressure, false scarcity, hidden continuity, or inflated claims.
