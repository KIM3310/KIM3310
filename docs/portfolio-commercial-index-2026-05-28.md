# Portfolio Commercial Index - 2026-05-28

This index is the final account-level map for the KIM3310 repository portfolio. It groups all public repositories by commercial lane and points reviewers toward the proof surface, monetization playbook, and safest next buyer conversation.

## Executive Read

The portfolio is strongest when read as a set of productized proof lanes:

- **Enterprise GenAI operations:** RAG, agents, governance, KPI dashboards, and controlled deployment.
- **Runtime reliability:** tool-calling, deterministic agent runtimes, orchestration, and evaluation gates.
- **Security and operations:** incident review, network assurance, local-first automation, and command-center surfaces.
- **Governed analytics:** data contracts, policy-aware analytics, finance/sector workflows, and audit evidence.
- **Applied AI products:** industrial inspection, regulated review, mobile/native surfaces, consumer experiments, and playable releases.
- **Enablement archives:** historical cookbooks, onboarding kits, and field playbooks that support workshops, partner enablement, and diagnostic offers.

4. Use `Nexus-Hive` and `lakehouse-contract-lab` for governed analytics proof.
5. Bring in archived repositories only as supporting material for workshops, partner enablement, or domain-specific context.
