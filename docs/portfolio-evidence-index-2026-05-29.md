# Portfolio Evidence Index - 2026-05-29

This index turns the full KIM3310 repository set into a reviewer-friendly proof map. Each active repository now has an Executive Proof Pack with a 7-minute evidence route and quality gate.

## How to Review the Portfolio

1. Start with the profile README and the live portfolio site.
2. Use this index to choose a lane: enterprise GenAI, runtime reliability, security/ops, governed analytics, industrial AI, mobile/product, or consumer funnels.
3. For each selected repository, read `docs/reviewer-evidence-map.md` before implementation files.
4. Use `docs/quality-gate.md` to confirm the verification and safety boundary.
